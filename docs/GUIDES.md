## 🎯 דשבורד

<a name="דשבורד"></a>

### מה זה?
הדשבורד המרכזי מציג את כל התחנות וסטטוס הלקוחות בזמן אמת.

### איך להשתמש?
1. פתח `http://localhost:5000/`
2. ראה את כל התחנות עם:
   - מספר לקוח בשירות כרגע
   - עד 10 מספרים המחכים בתור

### רענון
- אוטומטי כל 3 שניות
- ניתן ליעוץ ידני

### טיפים
- לא סולח כלום - מוצג רק סטטוס נוכחי
- הומלץ להציג על מסך גדול

## 👨‍💼 מפעיל

<a name="מפעיל"></a>

### מה הוא עושה?
מפעיל קורא ללקוח הבא בתור וסיימ שירות.

### כניסה
1. בחר תחנה
2. הזן קוד מפעיל (מ-config.json)
3. לחץ "התחבר"

### פעולות עיקריות

#### קרא ללקוח הבא
```
1. לחץ "קרא ללקוח הבא"
2. מספר הלקוח יופיע בצבע מדגיש
3. תצוגה תתרומם בדשבורד
```

#### סיים שירות
```
1. לחץ "סיום שירות"
2. לקוח הופך ל-"completed"
3. הלקוח הבא יוביל קוראו
```

#### הוספת לקוח למקום ספציפי
```
1. לחץ "הוסף מקום"
2. הזן מספר לקוח ומיקום
3. לקוח יוכנס בחזרה לתור
```

### דוגמה זרימה
```
1. מפעיל מתחבר: code="1001"
2. לחץ "קרא ללקוח הבא" → customer_number=101
3. לקוח 101 מקבל שירות
4. מפעיל לחץ "סיום" → customer_number=101
5. לקוח הבא: 102
```

## ➕ הוספת לקוחות

<a name="הוספה"></a>

### מי יכול להוסיף?
כל אחד בלא הגבלה

### איך מוסיפים?
1. בחר תחנה
2. הזן מספר לקוח (INT)
3. לחץ "הוסף"
4. לקוח יתווסף לסוף התור

### שגיאות נפוצות
- ❌ לקוח כבר בתור → בחר מספר אחר
- ❌ תחנה לא קיימת → בחר תחנה תקפה
- ❌ מספר לא תקפי → הזן מספר שלם

### נוספות
- Transfer: סמן אם לקוח מועבר מתחנה אחרת
- קבוצת תור: אם תחנה חולקת תור, הלקוח יתווסף לקבוצה

## 🔧 ניהול

<a name="ניהול"></a>

### גישה
1. בחר "ניהול" מתפריט ראשי
2. הזן סיסמה (admin123 כברירת מחדל)

### פעולות זמינות

#### צפה בכל הרשומות
- מסנן לפי תחנה
- מסנן לפי סטטוס
- הורדה כ-CSV

#### עדכון רשומה
- שנה סטטוס
- שנה תחנה
- שנה זמנים

#### מחיקת רשומה
- מחק לקוח מהתור
- מחק מהיסטוריה

#### דוח מפורט
- מספר לקוחות שהוגשו
- זמן ממתין ממוצע
- סטטיסטיקה לפי תחנה
- הורדה כ-PDF

#### הסתר תחנה
- לחץ על תחנה
- בחר "הסתר מדשבורד"
- תחנה לא תהיה גלויה למוזרים

## 🚨 בעיות נפוצות

| בעיה | פתרון |
|--------|--------|
| לקוח לא מופיע | בדוק תחנה נכונה |
| מפעיל קוד לא עובד | בדוק קוד ב-config.json |
| דשבורד לא מתרענן | F5 refresh / בדוק חיבור אינטרנט |
| לא ניתן להוסיף לקוח | אולי הוא כבר בתור? |
| דוח לא טוען | בדוק הרשאות admin |

## 💡 ניצול בטוב

### Performance
- אתרו תחנות משותפות (queue_group_id)
- הסתרו תחנות לא פעילות
- ראו דוח יומי בסוף יום

### Security
- שנו סיסמה admin
- סחפו קודי מפעילים
- צפו בלוג פעילויות

### Accuracy
- הזנו מספרים ייחודיים
- סמנו transfer כנדרש
- עדכנו זמנים אם שגוי
